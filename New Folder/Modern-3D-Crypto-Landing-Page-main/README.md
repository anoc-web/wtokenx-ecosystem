
CryptoX is a sleek, modern landing page template designed for cryptocurrency businesses, exchanges, and blockchain projects. Featuring cutting-edge 3D scrolling effects, smooth animations, and a dark theme with vibrant accents, this template is perfect for showcasing your crypto product or service.


Features
3D Scrolling Effects: Parallax layers and perspective transformations create depth

Modern Animations: Smooth fade-ins, hover effects, and interactive elements

Dark Theme: Sleek black design with neon green/blue accents

Responsive Layout: Fully responsive design for all devices

Interactive Elements: Animated token prices, hover cards, and more

Ready-to-Use Sections: Hero, features, stats, tokens, and CTA sections

Performance Optimized: Clean code for fast loading

Installation

No installation required! Simply:

Download the repo file

Open in any modern browser

Customize the content as needed

For hosting:

Upload the repo file to your web server

That's it! No additional dependencies needed

Customization
Changing Colors
Edit the CSS variables in the :root selector at the top of the file:

css
:root {
    --primary: #00ff88; /* Main accent color */
    --secondary: #00b3ff; /* Secondary accent */
    --dark: #121212; /* Main background */
    --darker: #0a0a0a; /* Darker elements */
    --light: #f8f9fa; /* Text color */
    --gray: #2a2a2a; /* Cards background */
}
Updating Content
Hero Section: Edit the content in the .hero section

Features: Update the feature cards in the #features section

Tokens: Modify the token cards in the #tokens section

Stats: Change the statistics in the #stats section

Footer: Update links and information in the footer

Adding Images
Replace the placeholder image URLs with your own:

html
<img src="your-image-url.jpg" alt="Your Image">
